A starter template for those who want to use the famous javascript 3D game engine babylonjs with typescript and webpack.

Customized from https://github.com/pandadelphin/babylonjs-typescript-webpack-starter

The included gltf model was made with photogrammetry using meshroom.

# About this branch

This branch is used to develop a prototypical implementation of an interaction system for VR.
It should handle physics objects, grabing them, throwing and finally click interactions.

Ideally this will be refactored into an extension for the VR Experience Helper.
Maybe some functionality can be merged into the VR Experience Helper at some point if it runs well enough? 

## How to Start? ##

1. Download and Install Node.js
2. Open Commandline and Enter
```
yarn install 
yarn start
```
3. Open Browser and Enter [http://localhost:8080](http://localhost:8080)

## How to build? ##
```
yarn run build
```
